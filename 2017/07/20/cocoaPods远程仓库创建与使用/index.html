cocoaPods远程仓库创建与使用

发表于 2017-07-20 | 分类于 ios

新建

1、创建一个专门用来存放spec文件的远程仓库

2.将远程库地址加入repo

pod repo add 名字XX 地址

3.创建本地模板库

pod lib create spec文件名称 //记得修改podspec文件

4.创建远程库 用于存放 步骤3 存放的库工程

5.验证spec文件

6.将spec 推送到步骤1的那个私有库

7.修改宿主工程podfile

8.update
pod update
或者pod repo update [远程仓库名]

9.安装
pod install
ps：在执行 pod install 之前最好先 rm Podfield.lock，保证能安装到最新的

维护（扩充代码，版本号升级）

需重新提交代码 修改spec描述文件，大概步骤如下：

  1. 重新提交代码

  1. 打tag 同时改spec描述文件

  2. 提交pec文件

  3. 更新spec文件到索引库
    pod repo push 名字XX spec文件名称.podspec
